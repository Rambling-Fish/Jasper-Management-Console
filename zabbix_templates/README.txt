Zabbix Templates 
=======================
Zabbix is a set of entities that can be conveniently applied to multiple hosts.
The entities may be:
•	items
•	triggers
•	graphs
•	applications
•	screens (since Zabbix 2.0)
•	low-level discovery rules (since Zabbix 2.0)
The available templates for Jasper are: 
JTA-Server.XML – includes all the monitored MBeans for Mule server that hosts the JTA’s and a set of generic alerts.
JSB.XML - includes all the monitored MBeans for ActiveMQ server and a set of generic alerts.
JTA-Generic.XML - includes all the monitored MBeans for Mule Applications (JTA) that runs the on the server and a set of generic alerts.
